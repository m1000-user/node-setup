# 🚀 Remnanode Auto-Installer

Автоматический Bash-скрипт для развертывания ноды Remnanode с проксированием через Nginx, установкой выбранного веб-сервиса и интеграцией Cloudflare WARP.

## 📋 Что делает этот скрипт?

Скрипт полностью настраивает сервер «с нуля»:

1. **Инфраструктура:** Установка Docker Engine и Docker Compose.
2. **SSL-сертификация:** Установка `acme.sh` и выпуск сертификатов **Let's Encrypt** (Standalone режим).
3. **Nginx Proxy:** Настройка прокси-сервера (порт `9443`) для безопасного HTTPS-доступа.
4. **Remnanode:** Установка ноды с автоматической подстановкой `SECRET_KEY` и путей к SSL-сертификатам.
5. **Безопасность:** Настройка Firewall (`ufw`) — открываются только необходимые порты.
6. **Cloudflare WARP:** Загрузка установщика WARP-клиента.

## 🛠 Подготовка

- **ОС:** Ubuntu 22.04+ или Debian 11+.
- **Домен:** A-запись должна указывать на IP сервера.
- **Права:** Запуск от имени **root** или через sudo.

## 🚀 Инструкция по запуску

1. Клонируйте репозиторий:
  ```bash
   git clone https://github.com/x1roko/node-setup.git
  ```
2. Перейдите в папку проекта:
  ```bash
   cd node-setup
  ```
3. Дайте права на выполнение:
  ```bash
   chmod +x install.sh
  ```
4. Запустите установку:
  ```bash
   ./install.sh
  ```
## 🚀 Установка одной командой

```bash
git clone https://github.com/x1roko/node-setup.git && cd node-setup && sudo chmod +x install.sh && sudo ./install.sh 
```

## ⌨️ Параметры ввода

Скрипт запросит:

1. **Email** (для регистрации SSL).
2. **Domain** (например, `node.domain.com`).
3. **SECRET_KEY** (ключ из панели управления Remnawave).
4. **Выбор сервиса** (введите номер или нажмите Enter для случайного выбора).

## 📂 Структура проекта

- `/opt/remnanode/` — Конфигурация ноды.
- `/opt/remnanode/nginx/` — Конфиги Nginx и SSL-ключи.
- `/opt/[название-сервиса]/` — Файлы выбранного веб-сервиса.

## ⚠️ Используемые порты


| Порт  | Сервис                          |
| ----- | ------------------------------- |
| 443   | xray (или по вашей настройке)   |
| 9443  | Веб-интерфейс (HTTPS)           |
| 2222  | Порт ноды Remnanode             |
| 40000 | Cloudflare WARP SOCKS5 (закрыт) |
