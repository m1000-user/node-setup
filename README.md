# 🚀 Remnanode Auto-Installer

Автоматический Bash-скрипт для развертывания ноды Remnanode с проксированием через Nginx, установкой веб-интерфейса FluffyChat и туннелем Cloudflare WARP.
## 📋 Что делает этот скрипт?

Скрипт полностью настраивает сервер «с нуля», выполняя следующие шаги:

1.  **Инфраструктура:** Установка Docker Engine и Docker Compose.
2.  **SSL-сертификация:** Установка `acme.sh` и выпуск сертификатов **Let's Encrypt** (через Standalone режим).
3.  **FluffyChat Web:** Развертывание веб-мессенджера в изолированном контейнере.
4.  **Nginx Proxy:** Настройка прокси-сервера (порт `9443`), который обеспечивает безопасный HTTPS доступ.
5.  **Remnanode:** Установка ноды с автоматической подстановкой вашего `SECRET_KEY` и путей к сертификатам.
6.  **Безопасность:** Настройка Firewall (`ufw`) — открываются только необходимые порты.
7.  **Cloudflare WARP:** Установка WARP-клиента и настройка SOCKS5 прокси на порту `40000`.

### 🛠 Подготовка

1    * **ОС:** Ubuntu 22.04+ или Debian 11+.

2    * **Домен:** Ваша A-запись должна указывать на IP сервера.

3    * **Права: Запуск от имени root.**

### 🚀 Инструкция по запуску

1. Клонируйте репозиторий:
```bash
git clone https://github.com/x1roko/node-setup.git
```

2. Перейдите в папку проекта:
```bash
cd node-setup
```

3. Дайте права на выполнение:
```bash
chmod +x install.sh
```

4. Запустите установку:
```bash
./install.sh
```

### ⌨️ Параметры ввода

Во время работы скрипт запросит:

1    **Email:** Для регистрации SSL.

2    **Domain:** Ваш домен (например, node.myserver.com).

3    **SECRET_KEY:** Ключ из панели управления Remnawave.

### 📂 Структура проекта

1.    **/opt/remnanode/** — Конфигурация ноды.

2    **/opt/remnanode/nginx/** — Конфиги Nginx и SSL ключи.

3    **/opt/fluffychat-web/** — Веб-чат.


### ⚠️ Используемые порты
Порт	Сервис
443     xray (или по вашей настройке)
9443	Веб-интерфейс (HTTPS)
2222	Remnanode Node Port
40000	Cloudflare WARP SOCKS5 (открывать не нужно)
